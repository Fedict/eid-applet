eID Applet Tests

Via this page you can test the eID Applet within your web browser.

eID Applet Identification This test will perform an eID identification using the eID Applet. A simple eID data readout will take place.

Click here to run the eID Applet Identification test.

 eID Applet Authentication This test will perform an eID authentication using the eID Applet. This test requires you to enter your eID PIN code.

Click here to run the eID Applet Authentication test.

 eID Applet ODF Signature This test will execute an eID Signature creation using the eID Applet. This test requires you to enter your eID PIN code. The demonstrated signature format is the Open Document Format (ODF). The ODF signature format is also supported by OpenOffice.org 3.2.

Click here to run the eID Applet Signature test.

 eID Applet OOXML Signature This test will perform an eID Signature creation via the eID Applet. This test requires you to enter your eID PIN code. The demonstrated signature format is Office OpenXML (OOXML). The OOXML signature format is also supported by MS-Office 2007.

Click here to run the eID Applet Signature test.
