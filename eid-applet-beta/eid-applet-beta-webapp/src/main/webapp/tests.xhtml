eID Applet Tests

Via this page you can test for correct functionality of the eID Applet within your web browser.

eID Applet Identification This test will execute an eID identification via the eID Applet. A simple eID readout will take place.

Click here to run the eID Applet Identification test.

 eID Applet Authentication This test will execute an eID authentication via the eID Applet. This test requires you to enter your PIN code.

Click here to run the eID Applet Authentication test.

 eID Applet Authentication - Kiosk Mode This test will execute an eID authentication via the eID Applet. This test requires you to enter your PIN code. The eID Applet kiosk mode features detection of eID card removal.

Click here to run the eID Applet Authentication (Kiosk Mode) test.

 eID Applet ODF Signature This test will execute an eID Signature creation via the eID Applet. The demonstrated signature format is ODF. The ODF signature format is also supported by OpenOffice.org version 3.1.

Click here to run the eID Applet Signature test.
